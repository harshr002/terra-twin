# TerraTwin

AI-powered digital twin platform for real-time IoT monitoring, anomaly detection, and live infrastructure intelligence.

---

# Live Demo

Backend API:

# What It Does

TerraTwin simulates IoT sensor streams, ingests real-time data using FastAPI, stores sensor readings in SQLite, detects anomalies using machine learning, and streams live updates to a React dashboard through WebSockets.
